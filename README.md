# MM plasma-secretory spatial program project

This repository is reserved for the code and compact reproducibility materials for the manuscript:

**Spatial validation and clinical association of a plasma-secretory bone marrow program in multiple myeloma**

Current target journal: BMC Medical Genomics.

## Claim boundary

The manuscript reports a reproducible and clinically associated plasma-secretory program in multiple myeloma. It does not claim completed R-ISS, PFS, treatment-response, prospective classifier, or wet-lab validation.
