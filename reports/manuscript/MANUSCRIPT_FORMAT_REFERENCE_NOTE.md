# Manuscript Format Reference Note

The current manuscript draft follows the structure of high-impact translational oncology and hematologic malignancy primary research articles:

- concise title;
- structured abstract with background, approach, core findings, and claim boundary;
- short mechanistic/translational introduction;
- reproducible public-data methods;
- figure-linked results;
- discussion that separates supported conclusions from limitations;
- explicit data availability and future-validation statements.

Reference class used for formatting only:

- Nature Genetics multi-omics cohort articles.
- Nature Cancer / Nature Medicine translational oncology articles.
- Cancer Cell translational tumor biology articles.
- Blood / Leukemia hematologic malignancy cohort articles.
- Nature Communications genomic cohort studies.

Excluded formatting references:

- textbooks;
- encyclopedia pages;
- blogs;
- non-peer-reviewed tutorials;
- low-quality papers;
- unverified template websites.

Formatting rule for this project:

- Do not overstate the claim.
- Do not present R-ISS, PFS, cytogenetic high-risk, or treatment-response validation as completed.
- Keep TXNDC5 as a spatial/single-cell localization candidate.
- Present `POU2AF1/XBP1/JCHAIN` and `plasma_secretory_score_z` as the clinically stronger subtype axis.
